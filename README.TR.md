# Mac'n'Minos

Dominos ® için çalışma

## Gerekenler

- Node > 13.2
- Yarn > ~

## Kurulum

    yarn

#### YADA

    yarn install

## Kullanım

APIyi json-server ile yaptım. APIyi ayağa kaldırmak için:

    yarn api

Not: Api Olmadan çalışmicaktır.

Development için: 

    yarn start
    
SSR için:

    yarn server
